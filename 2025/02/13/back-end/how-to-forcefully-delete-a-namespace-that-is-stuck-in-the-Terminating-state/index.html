дҪҶжҳҜеҸҜиғҪеӯҳеңЁеҲ йҷӨеӨұиҙҘжҲ–иҖ…namespaceдёҖзӣҙеӨ„дәҺTerminatingзҠ¶жҖҒзҡ„иҜқпјҢйӮЈд№ҲдёҠйқўзҡ„е‘Ҫд»ӨеҸҜиғҪиЎҢдёҚйҖҡгҖӮ
иҝҷйҮҢд»Ӣз»ҚдёӨз§Қе®һз”Ёзҡ„и§ЈеҶіж–№жЎҲеҺ»её®еҠ©жҲ‘д»¬и§ЈеҶігҖӮ

ејәеҲ¶еҲ йҷӨ

иҜҙжҳҺпјҡ

  • вҖ“forceпјҡејәеҲ¶еҲ йҷӨиө„жәҗпјҢи·іиҝҮжӯЈеёёзҡ„еҲ йҷӨжөҒзЁӢгҖӮ

  • вҖ“grace-period=0пјҡз«ӢеҚіеҲ йҷӨиө„жәҗпјҢдёҚзӯүеҫ…д»»дҪ•жё…зҗҶж“ҚдҪңгҖӮ

дҪҝз”ЁKubernetes API еҲ йҷӨ
