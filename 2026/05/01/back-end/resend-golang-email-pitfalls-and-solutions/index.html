一、背景

目标：

  • 使用 Golang 发送 HTML 邮件

  • 支持自定义域名发信（提升送达率）

二、Resend SMTP 基本认知（关键）

Resend 的 SMTP 和传统 SMTP 有一个重要区别：

类型 含义 示例
SMTP Username 登录用户名 resend（固定）
SMTP Password API Key re_xxx
From（发件人） 邮件发送地址 noreply@xxx.com

👉 重点：Username ≠ From

  • SMTP 要求符合 RFC 规范
