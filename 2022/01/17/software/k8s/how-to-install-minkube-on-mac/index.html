Mac版minikube安装笔记

minikube是一个专注于让Kubernetes更加容易学习和开发的本地Kubernetes。
只需要Docker或者虚拟机环境，我们便可以通过minikube start就能快速启动Kubernetes。

要求

安装minikube需要以下要求：

  1. 至少两个CPU
  2. 至少2GB内容
  3. 至少20GB的硬盘空间
  4. 容器或者虚拟机， 比如Docker, Hyperkit, Hyper-V, KVM, Parallels, Podman, VirtualBox, or VMware Fusion/Workstation
